# Baza_danych_biblioteki
Projekt obejmuje autorską aplikację konsolową typu DBMS (Database Management System), napisaną w języku C++. Narzędzie służy do ewidencji zasobów bibliotecznych, zarządzania statusem wypożyczeń oraz trwałego składowania danych. Program został zaprojektowany z myślą o szybkości działania i intuicyjnym interfejsie użytkownika (CLI).
